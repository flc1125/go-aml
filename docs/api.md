## [cite_start]1. API 规格：整批上传档案 [cite: 1, 2]

* [cite_start]**网址**: `https://aml.tdcc.com.tw/amlupload/api/uploadBatch` [cite: 3]
* [cite_start]**传输方式**: POST [cite: 4]
* [cite_start]**传输格式**: `form-data` 或 `x-www-form-urlencoded` [cite: 5]
* [cite_start]**编码**: UTF8 [cite: 6]
* [cite_start]**回传格式**: JSON [cite: 7]

### [cite_start]上传参数规格 (Request Body) [cite: 8, 9]

| 序号 | 参数名称 | 说明/值 | 栏位长度 |
| :--- | :--- | :--- | :--- |
| 1. | **userid** | 使用单位帐号 | 字串，最多 14 个字 |
| 2. | **password** | 使用单位密码 | 字串，最多 16 个字 |
| 3. | **fileDesc** | 上传档案说明 | 字串，最多 50 个字 |
| 4. | **file** | 上传档案 | 详见 P5【整批上传档案规格】 |

### [cite_start]回传规格 [cite: 10, 11]

| 序号 | 参数名称 | 类型 | 说明/值 | 栏位长度 |
| :--- | :--- | :--- | :--- | :--- |
| 1. | **status** | 必要 | **上传结果代码说明：**<br>• `ok`: 成功<br>• `error`: 失败 | 字串，最多 5 个字 |
| 2. | **batchNo** | 必要 | 上传资料成功会回传批号，使用此批号查询比对结果。<br>上传失败固定带 `ERROR` | 字串，最多 50 个字 |
| 3. | **errors** | 非必要 | **错误讯息阵列：**<br>• `row`: 上传档案行号<br>• `primaryKey`: 上传档案主键<br>• `errorMessage`: 错误讯息 | 阵列 |

## [cite_start]3. API 规格：单笔线上查询 [cite: 23]

* [cite_start]**网址**: `https://aml.tdcc.com.tw/amlupload/api/NameCheck` [cite: 24]
* [cite_start]**传输方式**: POST [cite: 25]
* [cite_start]**传输格式**: `form-data` 或 `x-www-form-urlencoded` [cite: 26]
* [cite_start]**编码**: UTF8 [cite: 27]
* [cite_start]**回传格式**: JSON [cite: 28]

### [cite_start]上传参数规格 (Request Body) [cite: 29, 30]

| 序号 | 参数名称 | 说明/值 | 栏位长度 |
| :--- | :--- | :--- | :--- |
| 1. | **userid** | 使用单位帐号 | 字串，最多 14 个字 |
| 2. | **password** | 使用单位密码 | 字串，最多 16 个字 |
| 3. | **NonEnglishName** | 非英文姓名，无资料为空字串 | 字串，最多 40 个字 |
| 4. | **EnglishName** | 英文姓名，无资料为空字串 | 字串，最多 80 个字 |
| 5. | **DOB** | 生日 (YYYY/MM/DD，如: 2017/09/01)，无资料为空字串 | 字串，固定 10 个字 |
| 6. | **Nationality** | 国籍，ISO 国家代码 (如台湾为 TW，美国为 US)，无资料为空字串 | 字串，固定 2 个字 |

### [cite_start]回传规格 [cite: 32, 33]

| 序号 | 参数名称 | 类型 | 说明/值 | 栏位长度 |
| :--- | :--- | :--- | :--- | :--- |
| 1. | **status** | 必要 | **上传结果代码说明：**<br>• `ok`: 成功<br>• `error`: 失败 | 字串，最多 5 个字 |
| 2. | **RCScore** | 必要 | 相似度分数，会回传 RC\>=91 或 RC=0。<br>如果上传结果失败，回传 `-1` | 整数，最多 3 个字 |
| 3. | **errors** | 非必要 | 错误讯息 | 字串，最多 50 个字 |

## 5\. CSV 档案规格说明

### [cite_start]5.1 整批上传档案规格 [cite: 49]

* [cite_start]**编码**: UTF8 [cite: 46]
* [cite_start]**上传格式**: CSV [cite: 47]

| 序号 | 参数名称 | 类型 | 说明/值 | 栏位长度 |
| :--- | :--- | :--- | :--- | :--- |
| 1 | **识别码** | 必要 | 上传档案唯一识别码 | 字串，最多 20 个字 |
| 2 | **非英文姓名** | 必要 | 非英文姓名 (两个其中一个一定要有值) | 字串，最多 40 个字 |
| 3 | **英文姓名** | - | 英文姓名 | 字串，最多 80 个字 |
| 4 | **生日** | 可选用 | 帐户所有人生日 (YYYY/MM/DD，如: 2017/09/01) | 字串，固定 10 个字 |
| 5 | **国籍** | 可选用 | ISO 国家代码 (如台湾为 TW，美国为 US) | 字串，固定 2 个字 |

> [cite_start]**注**[cite: 50, 51, 52]:
>
> 1.  栏位与栏位使用半形分号 `;` 分隔。
> 2.  国籍可以参阅 Wikipedia ISO 3166-1 alpha-2 中的 Code。

### [cite_start]5.2 整批上传比对结果档案规格 [cite: 55]

* [cite_start]**输出格式**: CSV [cite: 56]
* [cite_start]**编码**: UTF8 [cite: 56]

| 序号 | 参数名称 | 类型 | 说明/值 | 栏位长度 |
| :--- | :--- | :--- | :--- | :--- |
| 1 | **识别码** | 必要 | 上传档案唯一识别码 | 字串，最多 20 个字 |
| 2 | **最高 RC 分数** | 必要 | 名单资料比对的最高 RC 分数 | 整数，最多 3 个字 |
| 3 | **名单资料编号** | 必要 | 名单资料编号 | 字串，最多 10 个字 |
| 4 | **最高 RC 名单资料姓名** | 必要 | 名单资料匹配程度最高的姓名 | 字串，最多 350 个字 |
| 5 | **名单资料国籍** | 非必要 | Citizenship, Country of Registration Resident of. | 字串，最多 50 个字 |
| 6 | **名单资料生日** | 非必要 | DOB of the entity matched in watch list? | 字串，最多 50 个字 |
| 7 | **命中类型** | 非必要 | BL(黑名单)、NN(负面新闻)、RCA(亲属或亲密关系人)、PEP(高知名度政治人物) | 字串，最多 50 个字 |
| 8 | **名单最后审查日期** | 必要 | 名单最后审查日期 | 字串，最多 20 个字 |

> [cite_start]**注**[cite: 60, 61, 62]:
>
> 1.  栏位与栏位使用半形分号 `;` 分隔。
> 2.  系统仅取其中一笔分数最高者，请使用单位务必再进入洗钱防制查询系统，进一步查询更多资讯。

### [cite_start]5.3 客户批次比对结果档案规格 - 疑似洗钱通知单 (AML01) [cite: 63]

| 编号 | 栏位名称 | 资料型态 |
| :--- | :--- | :--- |
| 1 | **帐号/身分证字号/营利事业统编** | 字串 |
| 2 | **户名** | 字串 |
| 3 | **相似度分数** | 整数 |
| 4 | **名单资料姓名** | 字串 |
| 5 | **名单资料国籍** | 字串 |
| 6 | **名单资料生日** | 字串 |
| 7 | **命中类别** | 字串 |
| 8 | **名单编号** | 字串 |
| 9 | **名单最后审查日期** | 字串 |
| 10 | **备注** | 字串 |

> [cite_start]**注**[cite: 64, 66, 67, 68]:
>
> 1.  姓名模糊比对之相似度 RC (Relative Correlation) 值分数为 90，比对结果如有 \>RC 分数之情事，系统仅取其中一笔分数最高者；若 RC 值最高分有多笔时，取最高分名单中最后审查日期最新一笔。
> 2.  命中类别：BL(黑名单)、NN(负面新闻)、RCA(亲属或亲密关系人)、PEP(高知名度政治人物)。
> 3.  名单最后审查日期意指道琼斯于名单资料库有新增异动、修改、删除等情事所记载之日期或检视资料库之日期。
